# Python File Handling Assignment

## Overview

This project demonstrates basic **file handling operations in Python**.
The program performs multiple tasks such as writing data to files, reading data in different ways, appending new data, generating reports, and handling file errors safely.

The assignment is implemented using Python’s built-in file handling methods:

* `open()`
* `read()`
* `readline()`
* `readlines()`
* `write()`
* `append()`

---

## Tasks Implemented

### Task 1: Write Sales Records

A list of daily sales values is written to a file named **sales_data.txt**.

### Task 2: Read File in Different Ways

The program reads the sales file using three methods:

* `read()`
* `readline()`
* `readlines()`

### Task 3: Append New Sales

Additional sales values are appended to the existing **sales_data.txt** file.
The program then displays the updated file and counts the number of lines.

### Task 4: Generate Sales Summary

The program calculates:

* Total sales
* Highest sale
* Lowest sale
* Average sale

### Task 5: Product Information File

The user enters **3 product names and prices**.
These details are stored in **products.txt** and then displayed in a formatted output.

### Task 6: Safe File Reading

The user is asked to enter a filename.
The program checks whether the file exists before opening it.
If the file does not exist, an error message is shown.

### Task 7: Mini Project – Discount Report

A dictionary of product prices is used to:

* Apply a user-defined discount
* Calculate discounted prices
* Generate a report file called **discount_report.txt**

The report contains:

* Product name
* Original price
* Discounted price
* Summary statistics

---

## Project Files

The project folder contains the following files:

* `file_handling_assignment.py` – Main Python program
* `sales_data.txt` – Stores sales records
* `products.txt` – Stores product information entered by the user
* `discount_report.txt` – Generated discount report
* `README.md` – Project documentation

---

## How to Run the Program

1. Install Python (version 3.x recommended).
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

```
python file_handling_assignment.py
```

5. Follow the on-screen instructions to enter product details and discount percentage.
